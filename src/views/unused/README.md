- 避免直接删除可能有用的代码
- 保持主要代码库的整洁
- 方便将来需要时重新启用某些功能

## 🔄 重新启用指南

如果需要重新启用某个组件：

1. 将文件从 `unused/` 文件夹移回 `views/` 文件夹
2. 在 `router/index.js` 中添加相应的导入和路由配置
3. 在 `Dashbord.vue` 中添加相应的菜单项和路由处理
4. 测试功能是否正常工作

## ⚠️ 注意事项

- 移动到此文件夹的组件可能依赖于特定的API或服务
- 重新启用前请检查相关依赖是否仍然可用
- 某些组件可能需要更新以适配当前的项目结构
